## 🔧 Troubleshooting Highlights

<font size="4">

> 모든 트러블슈팅은 **검색해도 답이 안 나오는 영역**을 직접 부딪히며 해결한 경험입니다.

</font>

<table>
<tr>
<td width="50%" valign="top">

#### 🚨 [MUREAM] Mixed Content 차단 이슈

<font size="4">

**문제** — HTTPS 프론트엔드 ↔ HTTP 백엔드 사이의 모든 API 요청을 브라우저가 차단

**해결** — 인증서 발급 일정이 빠듯해서 **CloudFront Behavior 규칙으로 EC2 프록시 구성** → 단일 HTTPS 도메인으로 통일

**배운 점** — 풀스택 개발자가 가져야 할 시야는 *인프라 + 프론트 + 백엔드를 함께 보는 눈*

</font>

</td>
<td width="50%" valign="top">

#### 🤖 [MUREAM] AI 챗봇 모델 전환

<font size="4">

**문제** — Seq2Seq LSTM 300 epoch 학습 후에도 858개 QA로는 답변 품질이 일관되지 않음

**해결** — 생성형 → **검색형 모델로 구조 전환** TF-IDF 후보 20개 → KoBERT 의미 유사도 → 최종 선택 **하이브리드 구조 + 6개 카테고리 폴백 UX**

**배운 점** — 모델 선택은 무조건 최신 기술이 아니라 **데이터 양과 도메인 특성에 맞춰야 한다**

</font>

</td>
</tr>
<tr>
<td width="50%" valign="top">

#### 🧩 [Swachron] GLB 3D 모델 부품 자동 분리

<font size="4">

**문제** — 키크론 GLB 한 모델에 메시 수백 개 케이스/스위치/키캡 자동 판별 불가

**해결** — `worldMatrix` 누적 계산 + 면적 **mode 기반 분류 알고리즘 직접 작성** + 잘못 분류된 29개 수동 블랙리스트 = **자동 + 수동 하이브리드**

**성과** — 주요 5개 GLB / 196개 상품 분리 정확도 **100%**

</font>

</td>
<td width="50%" valign="top">

#### 🎯 [Swachron] WebSocket 핫딜 경매 구현

<font size="4">

**문제** — 키보드는 시간 제한 선착순 경매와 어울리지 않는 상품 특성. 팀 프로젝트에서 익힌 WebSocket 기술을 살리고 싶었지만, **이미 구현한 일반 경매 기능을 폐기해야 할 위기**에 봉착.

**해결** — 기능 폐기 대신 **비즈니스 컨셉을 전환**. *"한정판 키보드 중고 경매"* 로 재설계해 관리자가 **30~70% 할인율 + 상품 상태**를 선택 가능하게 구현 → WebSocket 실시간 호가는 그대로 유지.

**배운 점** — 기능을 버리지 않고 **컨셉을 바꿔서 살려내는** 제품 감각

</font>

</td>
